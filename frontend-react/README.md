## Quick Start

```bash
# Install dependencies
npm install

# Configure environment
cp .env.example .env.local
# Edit .env.local with your Keycloak settings

# Run development server
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser.
