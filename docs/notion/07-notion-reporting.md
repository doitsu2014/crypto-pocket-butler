# Notion Reporting (Dashboards)
## Databases to create
- Accounts: wallets/exchanges + metadata.
- Holdings snapshots: per account + consolidated (timestamped).
- Daily briefs: market regime, risks, suggested actions.
## Views
- Today: latest snapshot + top 10 holdings.
- Drift view: assets beyond drift threshold.
## Telegram outputs
- Daily: portfolio change %, biggest movers, 1-2 suggested actions.
