# MVP Scope & User Stories
## Primary user
- You (single user) managing multiple wallets + exchange accounts.
## MVP user stories
1. As a user, I can connect wallets/exchanges and see consolidated balances.
1. As a user, I can see allocation by asset and by account (wallet/exchange).
1. As a user, I get a daily brief: big moves, concentration, drift vs targets.
1. As a user, I receive rebalancing suggestions (no auto-trade in MVP).
## Non-goals (MVP)
- Automatic trading / withdrawals.
- Perfect cost basis/tax accounting (optional later).
## MVP deliverables
- [ ] Notion dashboard with portfolio + daily brief database
- [ ] Connect OKX (read-only) + 1 wallet type (EVM)
- [ ] Agent generates weekly rebalancing suggestion
